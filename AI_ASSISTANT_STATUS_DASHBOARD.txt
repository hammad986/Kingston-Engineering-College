For any page you want to add AI:

Step 1️⃣  Copy AI Widget HTML to <body>
   └─ Source: HEADER_TEMPLATE.html

Step 2️⃣  Copy AI Widget CSS to <style> in <head>
   └─ Copy all .ai-* styles

Step 3️⃣  Add Script Before </body>
